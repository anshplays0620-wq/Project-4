# Internship Project 4 – Interns Dashboard

A full‑stack mini application built with **Node.js + Express** (backend) and **HTML, CSS, JavaScript** (frontend).  
This project demonstrates frontend–backend integration, CRUD operations, and modern UI design.

---

## 🚀 Features
- **Backend (Express.js)**
  - RESTful API with CRUD routes (Create, Read, Update, Delete)
  - JSON data handling
  - CORS enabled for cross‑origin requests
  - Error handling with proper HTTP status codes

- **Frontend (HTML/CSS/JS)**
  - Responsive dashboard layout
  - Attractive card‑based UI for interns
  - Form to add new interns
  - Dynamic data rendering using `fetch()` with async/await
  - Error handling and fallback messages
